查看与更新linux系统python版本【详细！】

查看系统python版本并作相应修改

一、查看系统中的 Python 版本

  这里我的服务器系统是Ubuntu，以此为例子 ，命令如下：

  1. 使用 python3 命令：
    python3 --version

    我原本python的版本是3.8
  2. 使用 python3 命令加上 -V 选项：
    python3 -V
  3. 查看系统中所有已安装的 Python 版本：
    ls /usr/bin/python*

二、更新系统python版本

  这里以python==3.10为例子
  要更新 Ubuntu 系统中的 Python 版本，你可以按照以下步骤操作：

  1. 安装 Python 3.10：
    sudo apt update
sudo apt install python3.10
  2. 安装 Python 3.10 的 pip：
    sudo apt install python3.10-venv python3.10-dev
  3. 设置 python3.10 为默认版本：
    sudo update-alternatives --install /usr/bin/python3 python3 /usr/bin/python3.10 1
  4. 验证安装是否成功：
    python3 --version

    更改完成之后，我的系统python版本就是3.10了

  经过步骤将会在你的系统上安装 Python 3.10 并将其设置为默认的 Python 版本。
  请注意，这可能会影响到依赖于 Python 版本的其他应用程序，所以请确保你的系统中的所有应用程序都能正常运行在新的 Python 版本下。

三、遇到的报错

  执行$ sudo apt install python3.10的时候报错：

Building dependency tree       
Reading state information... Done
E: Unable to locate package python3.10
E: Couldn't find any package by glob 'python3.10'

原因、解决方法：
  在某些 Ubuntu 版本中，默认的软件源可能不包含 Python 3.10。你可以尝试通过添加软件源来解决这个问题。以下是在 Ubuntu 20.04 中添加软件源的步骤：

  1. 更新软件包索引：
    sudo apt update
  2. 安装软件源管理工具：
    sudo apt install software-properties-common
  3. 添加 deadsnakes PPA 软件源：
    sudo add-apt-repository ppa:deadsnakes/ppa
  4. 再次更新软件包索引：
    sudo apt update
  5. 安装 Python 3.10：
    sudo apt install python3.10

  这些步骤将会添加 deadsnakes PPA 软件源，使你可以安装 Python 3.10。完成后，你应该可以顺利安装 Python 3.10 了。
