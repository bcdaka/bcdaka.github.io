安装Android Studio报错unable to access android sdk add-on list的解决方案

1. 安装报错

初次安装Android Studio时，提示报错如下：

unable to access android sdk add-on lis

2. 报错原因

安装Android Studio时，会在默认路径下检测是否有Android SDK，如果没有的话，就会报上述错误。

3. 解决方案（亲测可行！）

3.1 选择Setup Proxy

  • 选择Manual proxy configuration

  • 设置 Host name 为：mirrors.neusoft.edu.cn

  • 设置 Port number 为：80

  • 点击Check connection

  • 提示success，成功解决
