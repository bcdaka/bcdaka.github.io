uniapp和微信小程序互相转换

✨一、uniapp转换为小程序

这个简单，HubuilderX自带的发行就可以转

🚩二、小程序转换为uniapp

小程序转uniapp工具安装方式（任选一种）

🎉1. 安装

1. 命令行安装方式：

npm install miniprogram-to-uniapp -g
2.HBuilderX插件：

uniapp插件市场：miniprogram-to-uniapp v2
小程序与uniapp的语法并不能一一对应，因此，仍然会有一些语法问题是工具层面无法解决的

🔑2. 使用

  • 这里用的是引入插件的方法
  • 打开插件链接：HBuilder X 插件 miniprogram-to-uniapp，点击右侧绿色按钮“使用HBuilderX导入插件”。

在HBuilder X左侧项目管理器里，在小程序项目上右键，点击弹出菜单里面的 “miniprogram to uniapp v2”，然后等待插件运行结束即可。

等待转换完成



转换完成后在小程序代码同源目录会有个uni结尾的文件夹，用Hbuilder打开即可，可能有部分bug需要自己手工改
