快速解决 adb server version doesn‘t match this client

这个问题是由于电脑上安装了多个版本的adb工具，客户端和服务端的版本不一致，无法正常通信导致。最快的解决方法就是将Android SDK中adb复制到系统目录下。

操作步骤如下：

1. 查看adb版本和路径

执行adb version，如下，Instanlled as 显示的路径就是本机的adb文件路径。

2. 覆盖adb

然后在你的电脑上找到对应版本的adb服务端文件（一般在Android SDK目录下的platform-tools文件夹中），执行sudo cp指令将其复制到系统路径中，覆盖掉当前版本的adb服务端文件 。
