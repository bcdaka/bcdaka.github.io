基于Python+FFMPEG环境下载B站歌曲

题主环境

WSL on Windows10

命令如下

# python3.9
pip install --pre yutto
yutto --batch https://www.bilibili.com/video/BV168411o7Bh --audio-only
ls | grep aac | xargs -I {} ffmpeg -i {} -acodec libmp3lame {}.mp3
