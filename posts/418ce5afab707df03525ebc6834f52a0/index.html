jar 命令启动java 指定配置文件路径 jar如何启动

可以使用以下命令来启动一个包含配置文件的jar包：

java -jar yourJarFile.jar --spring.config.location=file:/path/to/your/config/file/application.properties

其中，yourJarFile.jar是你的jar文件名，--spring.config.location指定了配置文件的路径。将/path/to/your/config/file/application.properties替换为你的配置文件的实际路径。

如果你的jar包没有使用Spring Boot框架，则需要使用合适的参数来指定配置文件路径。例如：

java -jar yourJarFile.jar -Dconfig.file=/path/to/your/config/file/application.properties

其中，-Dconfig.file参数指定了配置文件的路径。同样，将/path/to/your/config/file/application.properties替换为你的配置文件的实际路径。
