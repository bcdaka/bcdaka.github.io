🉑 我搭建了一个 AI Agent 自媒体团队，能自动写文章和发帖子

https://www.youtube.com/watch?v=4ZqJSfV4818

https://twitter.com/MatthewBerman/status/1750595382068023661

博主 @MatthewBerman 创建了一个 AI Agent 工具「AutoGen Studio 2.0」，它可以充当一个现实世界中的「自媒体团队」。

简单来说，就是给一个 Youtube 视频链接，它就可以自动获取字幕，然后生成播客文章、X平台的推文线程！！这也太方便了！！

演示视频的内容大致如下，可以看到作者开发和使用的完整流程：

  • 安装 AutoGen Studio，指定端口，并在浏览器的新标签页中访问最新版本

  • 在 AutoGen Studio 2.0 中创建和定制 Agent 团队，使用 OpenAI 的 chatGPT 技能，以及获取 YouTube 视频字幕

  • AutoGen Studio 2.0 允许在没有 API 密钥的情况下访问 YouTube 数据 API，并安装获取 YouTube 视频字幕的 API

  • 创建一个「获取 (fetch)」技能来获取 YouTube 视频字幕，然后创建 Agent 团队来去除时间戳，并把文本转换为博客文章和 X 平台的推文线程

  • 再次演示：创建一个新的工作流程，使用群聊让三个 Agent 协作处理 YouTube 视频脚本到内容的工作流程

  • AutoGen Studio 2.0 工作流串讲：如何设置多 Agent 团队来完成获取 YouTube 视频字幕、创建博客文章等任务，注意需要明确的描述

  • 工作流程完整展示：AutoGen Studio 2.0 允许创建多 Agent 团队并执行现实世界的任务，无需编写代码，包括解决 API 密钥问题和执行获取 YouTube 视频字幕、创建博客文章等任务



🉑 真正「千人千面」的时代到来了：GPT Newspaper 为每个人定制报纸

https://github.com/assafelovic/gpt-newspaper

GPT Newspaper 整个项目很有意思，可以根据每位用户给定话题偏好、新闻来源等关键词，生成一份专属的报纸！而且还支持选择排版样式~

所以，每位用户都可以根据自己的兴趣，来策划和设计一份独一无二的报纸了！！当前项目可以将生成的报纸发送到用户邮箱，如果将来有同类型的成熟产品后，AI内容完全可以实现「即产即销」~

哇哦！内容创作和消费即的个性化和千人千面时代，已经有隐约的轮廓了 🎉


根据给出的信息我们可以看，这也是一个 AI Agent 项目，并且清晰地设置了7个不同的角色和功能！看来 Agent 的设置还是需要行业 know-how 哇：

  1. 搜索 (Search)：在网络上搜索最新和最相关的新闻

  2. 策划 (Curator)：根据用户定义的偏好和兴趣筛选和选择新闻

  3. 撰写 (Writer)：撰写引人入胜、易于阅读的文章

  4. 批评 (Critique)：向撰写代理提供反馈，直到文章获得批准

  5. 设计 (Designer)：为文章布局和设计，提供美观的阅读体验

  6. 编辑 (Editor)：根据制作好的文章构建报纸

  7. 发布 (Publisher)：将报纸发布到前端或用户指定的服务器

👀 Cubox 创始人：不要把「总结全文」当做AI阅读产品的唯一亮点

https://cubox.pro

🔔 新出的AI功能是要付费才可以体验吼~

Cubox 是一款阅读器，有桌面应用、移动应用、浏览器插件等多多种产品形态。其实，Cubox很早便推出了AI助手，但因种种原因下线了相关功能。

最近，Cubox 推出了全新的基于大模型的 Cubox Insigh，以全新的面貌阐述了团队对于「AI+阅读」的更多理解：

不仅可以自动翻译、总结摘要、解读关键问题、标注、问答、深度分析等等，还应该支持深度交互：

  • 在阅读某段文字产生更多好奇心时，求助 AI 解释相关内容，或者提供扩展的阅读资料；
  • 遇到有趣的内容，让 AI 写一段分享文案，快速分享给好友；
  • 阅读过程中，有一些想法但不知道如何组织，让 AI 帮助写一篇阅读笔记大纲，再去高亮笔记中笔记编辑，方便未来回顾……

当下AI阅读助手最常见的功能，就是总结文档/文章摘要、有些还能给出思维导图。但是 Cubox 与这些都不太相同，它向前走了一步，把阅读最常用的各类功能，套进了一个AI的大壳里！！

Cubox 创始人也随更新发了一篇文章，把他日常碎片化的思考整理成文，探寻AI 对阅读产生的影响，兼具了人文关怀和思考深度

AI 不会让阅读消失，AI 提供的新的阅读方式，也并非新的阅读哲学，而是传统阅读经验在新技术下，在更少的人工的参与下，更好的兑现 ⋙ 非常推荐阅读原文
