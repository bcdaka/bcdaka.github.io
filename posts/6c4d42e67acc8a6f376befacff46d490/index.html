1008：计算(a+b)/c的值

时间限制: 1000 ms 内存限制: 66536 KB
提交数:164836 通过数: 142434
【题目描述】
给定3个整数a、b、c，计算表达式a+bc的值。

【输入】

输入仅一行，包括三个整数a、b、c, 数与数之间以一个空格分开。(－10,000<a,b,c<10,000, c不等于0)

【输出】

输出一行，即表达式的值。

【输入样例】

1 1 3

【输出样例】

0

#include<iostream>
using namespace std;
int main() {
	int a,b,c;
	cin>>a>>b>>c;
	cout<<(a+b)/c<<endl;
	return 0;
}
