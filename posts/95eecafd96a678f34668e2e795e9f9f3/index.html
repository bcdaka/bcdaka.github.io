如何在 Mac 上重置网络设置

Mac 几乎在所有时间都非常可靠，但有时您在连接到互联网时可能会遇到困难或浏览速度缓慢。 互联网可能在您的其他设备上正常工作，这可能很烦人。 通常，问题的原因是什么并不明显，甚至根本不存在。

如果您发现自己处于类似情况，您可能需要重置 Mac 上的网络设置。 您可以重置各种设置，细节可能令人生畏。 不要害怕; 我们将解释您应该尝试解决什么网络问题。

为什么您可能需要重置 Mac 上的网络设置？

通常，您应该对使用 macOS 充满信心。 这是一个可靠的操作系统，在 99% 的时间里都是可靠的。 但是，有时您可能会发现自己无法连接到 Internet 或浏览网站。 Wi-Fi 工作正常，互联网已在您的所有其他设备上正常运行，但由于某种原因，您的 Mac 无法连接。

如果您经常使用 VPN，则您的系统上的网络设置可能会被弄乱。 Internet 设置可能非常敏感，因为即使是最轻微的更改或调整也可能导致与 Internet 的连接中断。

如何在 Mac 上重置 Internet 设置

好消息是，与其他操作系统一样，您可以重置 Mac 上的网络设置。 不幸的是，有一些坏消息。 与 Windows 甚至 iOS 不同，Mac 上没有一键重置网络设置的方法。

您可以使用两种解决方案来重置 Mac 的网络设置。 如果您遇到临时问题，只需重置当前连接的 Wi-Fi 网络即可。 您的 Mac 将忘记当前网络的设置，您可以从头开始连接，就像您第一次设置时一样。 这样做通常会重置 Wi-Fi 网络设置，并且应该可以解决问题。

但是，如果问题仍未解决，您可以使用第二种方法从 Mac 中删除网络配置文件。 这是一种复杂的方法，但它是解决您可能面临的任何顽固网络问题的最终方法。

在继续之前，请确保问题出在 Mac 上，而不是出在 Wi-Fi 连接本身上。 一个简单的方法是检查路由器上的互联网连接指示灯。 您还应该检查互联网在连接到同一 W-Fi 网络的任何其他设备（iPhone、iPad 等）上是否正常工作。

如何重置 Wi-Fi 连接

第一种也是更简单的方法是重置 Wi-Fi 或以太网的网络连接。 这将删除连接，然后重新建立它。 它还会从您的系统中删除 Wi-Fi 数据（用户名和密码），因此您必须记下详细信息。 这将帮助您稍后重新创建连接。

以下是重置 Wi-Fi 连接的方法。 这是使用的默认连接，也是最常见的连接，但您的连接可能不同。 您可能正在使用以太网连接，因为 example.

  1. 打开 系统偏好设置 在您的 Mac 上。
  2. 点击 网络.
  3. 选择 无线上网 从左侧窗格。
  4. 点击 减 (-) 连接列表下方的图标。
  5. 点击 申请. 这应该会禁用 Mac 的 Wi-Fi。
  6. 现在，要再次添加它，请选择 加号 (+) 图标。
  7. 选择 无线上网 并选择 创建.
  8. 点击 申请，您应该会看到选择 Wi-Fi 连接的选项。 选择要连接的连接并输入密码。 您的 Mac 现在应该可以连接，并且互联网应该可以正常工作。

重置网络配置文件

如果上面详述的方法对您不起作用，那么这个方法应该会有所帮助。 这有点复杂，但它是完全安全的。 此过程会删除与网络设置相关的某些系统文件。 您的 Mac 将在您下次重新启动时恢复它们。

去做这个：

  1. 从顶部菜单栏中的 Wi-Fi 图标关闭 Wi-Fi。
  2. 打开 发现者，然后单击 走 在左上角的菜单栏中。 选择 转到文件夹.
  3. Enter 将以下地址输入文本字段并单击 走./图书馆/首选项/系统配置/
  4. Finder 窗口应该会打开，并在您面前显示一些系统文件。 选择以下文件：
    • com.apple.airport.preferences.plist
    • com.apple.network.identification.plist 或者 com.apple.network.eapolclient.configuration.plist
    • com.apple.wifi.message-tracer.plist
    • 网络接口.plist
    • 首选项.plist
  5. 复制这些文件并将它们保存到安全位置作为备份。 这将帮助您在遇到任何问题时恢复网络设置。
  6. 现在从 系统配置 文件夹。 选择它们并按 命令 + 删除 将它们移至废纸篓。
  7. 完成后，清空垃圾箱以永久删除文件。
  8. 现在重新启动您的 Mac。

macOS 现在应该重新生成您删除的网络配置文件，并且应该重置所有网络设置。

尝试连接到互联网，看看是否一切正常。 如果一切顺利，并且您可以成功浏览互联网，则可以继续删除之前作为备份保存的文件。

使用终端重置网络设置

如果您更习惯于使用终端而不是系统偏好设置应用程序，您可以使用它来重置网络设置。 这将重置当前的 Wi-Fi 网络并允许您重新加入它，类似于您在第一部分中所做的。

  1. 打开 终端 在您的 Mac 上。

  2. Enter 以下命令：

    sudo ifconfig en0 down

    您需要输入密码，因为 sudo 是特权命令。 输入后，该命令应关闭您的 Wi-Fi。

  3. 现在输入以下命令以重新启用 Wi-Fi：

    sudo ifconfig en0 up

  4. 就是这样。 再次尝试连接到 Wi-Fi 网络，看看它是否有效。
