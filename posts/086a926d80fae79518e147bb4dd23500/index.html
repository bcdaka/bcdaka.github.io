Android Studio安装SDK时报错：SDK emulator directory is missing

问题描述：

安装Android Studio时，一般情况下，Android Studio会自动配置环境变量。

但是在安装过程中首先会遇到Andriod SDK unavailable然后报了No Android SDK found 异常，最后出现SDK emulator directory is missing的错误。

问题说明：

根据提示明显缺少Android SDK，可单独下载SDK，然后配置路径；也可按照如下配置下代理，自动更新SDK来解决。

解决方法：

  1. 在错误界面，击右下角 finish 退出当前安装 SDK 的界面。
  2. 页面会自动跳转到android studio的正常页面
  3. 点击创建新项目，会提示没有安装SDK，点击Open SDK Manager
  4. 可以看到跳转到如下页面，点击Appearance & Behavior
  5. 如图选择System Settings当中的HTTP Proxy,选择Auto-detect proxy settings,输入配置路径：socks5h://127.0.0.1:1080 点击确定
