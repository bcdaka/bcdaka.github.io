第11章 Database

  1. data integrity（数据完整性）
  2. data redundancy（数据冗余）

-

  1. Data warehouse（数据仓库）：由组织从各种外部和内部的数据库收集的信息存储在这种类型的数据库。
  2. hierarchical database（层次数据库）：每个条目都有一个父节点，父节点有多个子节点。
  3. Object-Oriented database（面向对象的数据库）：不仅存数据，还存操作数据的指令。
  4. multidimensional database（多维数据库）：也叫data cube（数据立方体），将数据视为三面或更多面得到立方体。
  5. company database（企业数据库）：需要通过连接到局域网的个人电脑进行访问。

-

  1. real-time processing（实时处理）
  2. batch processing（批处理）：每月银行账单用的这个。
  3. group processing（分组处理）

  1. LexisNexis（词汇）：提供法律、商业、新闻等多领域信息检索服务的数据库。
  2. ProQuest Dialog：提供全球性学术文献的信息检索平台。
  3. CSI：用于法律法规方面的检索。
  4. Dow Jones Factiva：提供全球商业和新闻信息的数据库服务。

第12章 Syetem Analysis and Design

  1. Grid chart（网格图）：显示输入输出文档之前的关系。
  2. System audit（系统审计）：确定新程序是否实际提高了生产力。
  3. System design report（系统设计报告）：展示了成本与收益，并概述了替代设计对组织的影响。
  4. system analysis report（系统分析报告）：描述了当前的信息系统，新系统的需求以及可能得发展时间表。

-

系统生命周期

  1. Phase 1:Preliminary Investigation（初步调查）：提出替换解决方案。
  2. Phase 2:System Analysis
  3. Phase 3:systems design：设计新的或替代的信息系统。
  4. Phase 5:system implementation（系统实现）：另一个名字叫conversion（转换），在此阶段，安装了新的系统，用户也接受了新系统的培训。
  5. Phase 6:maintenance（维修）：大多数系统在该阶段花费时间或金钱更多，被认为是一场持续的活动。

-

  1. 系统的直接实现方法被认为是最有风险的。
  2. 并行系统实行方法仅在失败或中断操作的成本大的情况下使用。
  3. 系统分析和设计是检查改进信息系统的手段（程序）。
  4. Operational feasibility（操作可行性）：确定提议的系统是否实际可以运行。

第13章 Programming and languages

  1. top-down program design（自上而下的软件设计）
  2. Program specifications document（项目规范文档）
  3. 程序维护领域的活动分为两类：operations 和 changing needs。

-

  1. pseudocode（伪代码）
  2. symbol（符号）
  3. debug（调试）
  4. syntax error（语法错误）：违反编程语言的错误。

-

  1. Beta testing（β测试）：由一组选定的潜在用户对程序进行测试。

-

  1. Loop structure（回路结构）：while函数之类的结构。
  2. Program flowchart（程序流程图）：使用符号显示步骤的编程工具。
  3. compiler（编译器）：将源代码转换为机器语言代码。
  4. interpreter（解释器）：将过程语言一次一条转换并运行成机器代码。
  5. application generator（应用生成器）：提供预编写代码模块的程序。

-

  1. first-generation languages（第一代编程语言,1GLs）：难用的一个原因是根据计算机的不同而不同。
  2. second-generation languages（第二代编程语言,2GLs）：使用缩写和助记符，这些都会自动转换成01串。
  3. third-generation languages（第三代编程语言,3GLs）：也就是procedural（程序性的），只在表达可以解决一般问题的逻辑。
  4. fifth-generation languages（第五代编程语言，5GLs）:结合了人工智能的概念。
  5. OOP（Object-Oriented program，面向对象语言）
