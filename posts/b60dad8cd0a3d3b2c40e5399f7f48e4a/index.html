java excel读写工具栏EasyExcel集成与使用

EasyExcel 是一个基于 Java 的简单、快速、内存开销小的读写 Excel 工具，非常适合处理大量数据的场景。

下面是如何在项目中集成 EasyExcel 并使用它来读写 Excel 文件的基本步骤：

1. 添加依赖

        首先，在 Maven 项目的 pom.xml 文件中添加 EasyExcel 的依赖：

<dependencies>
    <dependency>
        <groupId>com.alibaba</groupId>
        <artifactId>easyexcel</artifactId>
        <version>3.0.5</version> <!-- 请根据最新版本调整 -->
    </dependency>
</dependencies>

2. 创建数据模型类

        EasyExcel 需要一个 Java Bean 来映射 Excel 表格中的每一行数据。

        例如，创建一个 User 类：

public class User {
    private String name;
    private String email;
    private Integer age;

    // Getter 和 Setter 方法
}

3. 写入 Excel

        创建一个工具类 ExcelUtil，用于操作 Excel 文件的读写：

import com.alibaba.excel.EasyExcel;

public class ExcelUtil {

    /**
     * 写入 Excel 文件
     */
    public static void writeExcel(String filePath) {
        // 指定写操作的文件路径
        EasyExcel.write(filePath, User.class)
                .sheet("用户信息")
                .doWrite(data()); // 提供数据源
    }

    /**
     * 数据源
     */
    private static List<User> data() {
        List<User> list = new ArrayList<>();
        for (int i = 0; i < 10; i++) {
            User user = new User();
            user.setName("User" + i);
            user.setEmail("user" + i + "@example.com");
            user.setAge(i + 20);
            list.add(user);
        }
        return list;
    }
}

4. 读取 Excel

        在 ExcelUtil 中添加读取 Excel 的方法：

/**
 * 读取 Excel 文件
 */
public static void readExcel(String filePath) {
    EasyExcel.read(filePath, User.class, new DataListener()).sheet().doRead();
}

/**
 * 数据监听器，处理读取到的数据
 */
private static class DataListener extends AnalysisEventListener<User> {
    @Override
    public void invoke(User user, AnalysisContext context) {
        System.out.println("姓名：" + user.getName() + ", 邮箱：" + user.getEmail() + ", 年龄：" + user.getAge());
    }

    @Override
    public void doAfterAllAnalysed(AnalysisContext context) {
        // 所有数据解析完成后执行的操作
    }
}

5. 使用工具类

        最后，在主方法或其他需要的地方调用这些工具方法来写入或读取 Excel 文件：

public static void main(String[] args) {
    String filePath = "D:/users.xlsx";
    ExcelUtil.writeExcel(filePath); // 写入数据到 Excel
    ExcelUtil.readExcel(filePath); // 从 Excel 读取数据
}
