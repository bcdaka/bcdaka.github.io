【快捷键】Mac如何打出数学符号和希腊字母

下面是一些常见的希腊字符及其对应的快捷键：

  1. α (Alpha): Option + A
  2. β (Beta): Option + B
  3. γ (Gamma): Option + G
  4. δ (Delta): Option + D
  5. ε (Epsilon): Option + E
  6. ζ (Zeta): Option + Z
  7. η (Eta): Option + H
  8. θ (Theta): Option + Q
  9. ι (Iota): Option + I
  10. κ (Kappa): Option + K
  11. λ (Lambda): Option + L
  12. μ (Mu): Option + M
  13. ν (Nu): Option + V
  14. ξ (Xi): Option + X
  15. ο (Omicron): Option + O
  16. π (Pi): Option + P
  17. ρ (Rho): Option + R
  18. σ (Sigma): Option + S
  19. τ (Tau): Option + T
  20. υ (Upsilon): Option + U
  21. φ (Phi): Option + F
  22. χ (Chi): Option + C
  23. ψ (Psi): Option + Y
  24. ω (Omega): Option + W

按住"Option"键，然后按下对应的字母键即可输入相应的希腊字符。

下面是一些常见的数学符号及其对应的快捷键：

加号 (+): Shift + =
减号 (-): Hyphen (-) 或者 Option + Hyphen (-)
乘号 (×): Option + Shift + 8
除号 (÷): Option + /
等于号 (=): Option + =
不等于号 (≠): Option + =
小于等于号 (≤): Option + <
大于等于号 (≥): Option + >
正无穷 (∞): Option + 5
逗号 (,) (用于表示小数点): Shift + ,
小数点 (.): 可直接键入
求和符号 (∑): Option + W
积分符号 (∫): Option + B
求导符号 (d/dx): Option + Shift + D
开方符号 (√): Option + V
平方根符号 (²): Option + 2
立方根符号 (³): Option + 3
