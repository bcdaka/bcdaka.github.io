使用pnpm创建vue3项目

https://pnpm.io/zh/

全局安装：

npm install -g pnpm

检查版本：

pnpm -v

创建vue3项目：

pnpm create vue@latest

项目装包：

pnpm install

运行项目：

pnpm dev

命令行：

https://pnpm.io/zh/pnpm-cli
 

pnpm install

pnpm add <pkg>

pnpm <cmd>
