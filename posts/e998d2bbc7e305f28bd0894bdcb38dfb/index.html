解决方案

  1. 将gradle版本更新
    打开根目录的build.gradle，修改gradle的版本为7.5以上即可。
    buildscript {
    repositories {
        google()
        mavenCentral()
    }
    dependencies {
    	/// 修改这里的版本号
        classpath "com.android.tools.build:gradle:7.5"

        // NOTE: Do not place your application dependencies here; they belong
        // in the individual module build.gradle files
    }
}
  2. 配置jvm
    打开项目中的gradle.properties, 在org.gradle.jvmargs的值后面追加--add-opens java.base/java.io=ALL-UNNAMED，重新编译即可
