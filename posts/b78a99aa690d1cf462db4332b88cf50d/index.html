一、时间和季节

Time and Season

1、时间描述

Time Description

比如，日出、黄昏、夜晚、清晨 / Sunrise, Sunset, Night, Early Morning

2、季节变化

Seasonal Changes

比如，春天、夏天、秋天、冬天 / Spring, Summer, Autumn, Winter

二、场景描述

Scene Description

1、自然景观

Natural Landscapes

比如，山脉、河流、森林、海滩 / Mountains, Rivers, Forests, Beaches

2、城市景观

Urban Landscapes

比如，天际线、街道、建筑、公园 / Skylines, Streets, Buildings, Parks

3、室内场景

Indoor Scenes

比如，家庭环境、办公室、咖啡馆 / Home Environment, Office, Café

4、抽象和概念性场景

Abstract and Conceptual Scenes

比如，梦境、虚拟空间、未来世界 / Dreams, Virtual Spaces, Future Worlds

三、物体和元素

Objects and Elements

1、自然元素

Natural Elements

比如，植物、动物、水、岩石 / Plants, Animals, Water, Rocks

2、人工物品

Man-made Objects

比如，建筑、家具、机械、交通工具 / Buildings, Furniture, Machinery, Vehicles

3、抽象元素

Abstract Elements

比如，几何形状、符号、图案 / Geometric Shapes, Symbols, Patterns

四、艺术风格

Artistic Style

1、传统艺术风格

Traditional Art Styles

比如，现实主义、印象主义、表现主义 / Realism, Impressionism, Expressionism

2、现代艺术风格

Modern Art Styles

比如，抽象艺术、波普艺术、极简主义 / Abstract Art, Pop Art, Minimalism

3、中国艺术风格

China Cultural Styles

比如，东方艺术、民间艺术、特定国家或地区风格 / Oriental Art, Folk Art, Regional Styles

4、摄影艺术风格

Photography Art Styles

比如，人像摄影、风景摄影、街头摄影、纪实摄影 / Portrait Photography, Landscape Photography ,Street Photography, Documentary Photography 

五、色彩和光线

Color and Light

1、色彩描述

Color Description

比如，鲜艳、柔和、冷暖色调 / Vibrant, Soft, Warm and Cool Tones

2、光线效果

Lighting Effects

比如，阳光、阴影、反射、透光 / Sunlight, Shadows, Reflection, Transparency

六、情感和氛围

Emotion and Atmosphere

1、情感表达

Emotional Expression

比如，快乐、悲伤、宁静、紧张 / Happiness, Sadness, Serenity, Tension

2、氛围营造

Atmospheric Creation

比如，神秘、浪漫、怀旧、梦幻 / Mysterious, Romantic, Nostalgic, Dreamy

七、构图和视角

Composition and Perspective

1、构图方式

Composition Techniques

比如，对称、非对称、层次、焦点 / Symmetry, Asymmetry, Layers, Focus

2、视角选择

Perspective Choices

比如，鸟瞰、仰视、正面、侧面 / Bird's-eye View, Low Angle, Front, Side View

八、人物与肖像

Characters and Portraits

1、人物特征

Character Features

比如，表情、姿态、装束、年龄 / Expression, Posture, Attire, Age

2、肖像风格

Portrait Styles

比如，写实、卡通、抽象、概念 / Realistic, Cartoon, Abstract, Conceptual

九、天气和气候

Weather and Climate

1、天气条件

Weather Conditions

比如，晴朗、多云、雨天、雪天 / Clear, Cloudy, Rainy, Snowy

2、气候特征

Climate Features

比如，炎热、寒冷、湿润、干燥 / Hot, Cold, Humid, Dry

十、质地和材料

Texture and Materials

1、表面质地

Surface Textures

比如，光滑、粗糙、柔软、硬质 / Smooth, Rough, Soft, Hard

2、材料类型

Material Types

比如，金属、木材、玻璃、石材 / Metal, Wood, Glass, Stone

十一、动作和活动

Actions and Activities

1、物体动作

Object Actions

比如，流动、旋转、跳跃、飞翔 / Flowing, Rotating, Jumping, Flying

2、人物活动

Human Activities

比如，跑步、跳舞、攀爬、游泳 / Running, Dancing, Climbing, Swimming
