mysql配置

3.1mysql下载

  1. 下载地址：MySQL

  2. 点击 MySQL Community(GPL) Downloads

  3. 点击 MySQL Community Server

3.2mysql安装

  1. 双击下载的mysql-installer-community-8.0.29.0.msi文件，开始安装。

  2. 打开“Choosing a Setup Type”（选择安装类型）窗口，选择“Custom（自定义安装）”类型按钮，单击“Next(下一步)”按钮。

  3. 打开“Select Products” （选择产品）窗口，可以定制需要安装的。例如，选择“MySQL Server 8.0.26-X64”后，单击“→”添加按钮，即可选择安装MySQL服务器，如下图所示。

  4. mysql安装路径（自行配置）

3.3mysql的配置

  1. 安装之后，需要对服务器进完成安装后，单击“Next”（下一步）按钮，就可以进入产品配置窗口。 “Config Type”选项用于设置服务器的类型，非特殊需要选第一个就好。

  2. 打开设置服务器root超级管理员的密码窗口，需要输入两次同样的登录密码。也可以通过“Add User”添加其他用户。

  3. 打开设置服务器名称窗口。将服务名设置为“MySQL80”。如果希望开机自启动服务，也可以勾选“Start the MySQL Server at System Startup”选项（推荐）。

3.4mysql配置环境变量

  1. 打开系统信息—>高级系统设置

  2. 打开环境变量—>系统变量

  3. 将mysql/bin目录下的地址复制

  4. 在path下添加路径

  5.  在cmd中输入mysql -uroot -p 输入密码（显示以下配置成功）
