AndroidStudio打包失败Error: Google Play requires that apps target API level 31 or higher.解决办法

这个问题是Android Studio和SDK升级后出现的，在打包apk时会出现报错。

首先找到build.gradle文件

在targetSdkVersion上面加一行注释：

//noinspection ExpiredTargetSdkVersion

然后在Android{}块中加入 

lintOptions {
    abortOnError false
}

再次打包就ok了
