Ps：首选项 - 文件处理

Ps菜单：编辑/首选项

Edit/Preferences

快捷键：Ctrl + K

Photoshop 首选项中的“文件处理” File Handling选项卡允许用户精确控制 Photoshop 的文件保存行为和兼容性选项。这些设置非常重要，尤其在处理大文件或与其他软件协作时，可以优化工作流程并确保文件的兼容性和安全性。

文件存储选项

File Saving Options

图像预览

Image Previews

设置在保存文件时是否将图像缩览图一起保存。默认为“总是存储”。

附带了缩览图的文件要比没带缩览图时稍微大一些。

本选项实质上控制着“存储为”和“存储副本”对话框中“缩览图”选项的状态。

--总是存储 

Always Save

如上图所示，默认勾选，且不可取消。

--总不存储

Never Save

不生成缩览图，且“缩览图”选项不可勾选。

--存储时询问

Ask When Saving

在“存储为”对话框中，可自行决定是否勾选生成“缩览图”。

文件扩展名

File Extension

选择文件保存时使用的小写或大写的文件扩展名。通常选择小写以符合现代操作系统的惯例。

--使用小写

Use Lower Case

--使用大写

Use Upper Case

存储至原始文件夹

Save As to Original Folder

默认勾选，执行存储命令时，将自动选择文件的原始文件夹中进行保存。

若不勾选，执行存储命令时，将自动选择上一次保存的文件夹进行保存。

后台存储

Save in Background

默认勾选，文件将以后台进程方式保存，使您可以继续在 Photoshop 中工作，而不会因为保存操作而中断工作。

若不勾选，则当保存文档时，我们无法进行其他操作，直至保存完成。

自动存储恢复信息的间隔

Automatically Save Recovery Information Every 

默认为 10 分钟 10 Minutes，表示一旦 Photoshop 意外崩溃，再次启动时可恢复文档到 10 分钟之前的状态。

可以选择从 5 分钟到 1 小时不等。

启用旧版“存储为”

Enable legacy "Save As"

默认不勾选。

勾选后，Photoshop 将使用旧版的“存储为”对话框界面。

在储副本时不将“副本”追加到文件名

Do not append "copy" to filename when saving a copy

默认不勾选。

勾选后，当“存储副本”时，不会自动在文件名后添加“拷贝”字样。

文件兼容性

File Compatibility

Camera Raw 首选项

Camera Raw Preferences

点击此按钮可以打开 Adobe Camera Raw 的首选项窗口，用于设置 Raw 格式文件的处理选项。

请参阅：

《Camera Raw：首选项》

对支持的原始数据文件优先使用 Adobe Camera Raw

Prefer Adobe Camera Raw for Supported Raw Files

默认勾选。Photoshop 会自动使用 Adobe Camera Raw 来打开支持的 Raw 格式文件。

事实上，不论此项是否勾选，当使用 Photoshop 打开 Raw 格式文件时，都会启动 Adobe Camera Raw。

使用 Adobe Camera Raw 将文档从 32 位转换到 16/8 位

Use Adobe Camera Raw to Convert Documents from 32 bit to 16/8 bit

默认不勾选。

若勾选，Photoshop 会自动使用 Camera Raw 将 32 位图像转换为 16 位或 8 位图像。

忽略 EXIF 配置文件标记

Ignore EXIT Profile Tag

默认不勾选，表示保留图像 EXIF 指定的颜色配置文件。

若勾选，Photoshop 将忽略图像文件中的 EXIF 配置文件标记，而使用默认的颜色配置文件或者不再指定颜色配置文件。

忽略旋转元数据

Ignore Rotation Metadata

默认不勾选。照片元数据中记录了拍摄时是否旋转相机的信息（比如，横拍、竖拍等）。

若勾选，Photoshop 将忽略图像文件中的旋转元数据，而按标准方向打开图像。

存储分层的 TIFF 文件之前进行询问

Ask Before Saving Layered TIFF Files

默认勾选，将带图层的文档保存为 TIFF 格式时，Photoshop 会弹出如下所示的对话框。

若不勾选此项，则不再弹出此对话框。

停用 PSD 和 PSB 文件压缩

Disable Compression of PSD and PSB Files

默认不勾选。

若勾选，Photoshop 将不会对 PSD 和 PSB 文件进行压缩保存，这将增加文件大小，但文件的保存和打开速度会更快。

最大兼容 PSD 和 PSB 文件

Maximize PSD and PSB File Compatibility

选择保存 PSD 和 PSB 文件时是否启用最大兼容性，这通常用于确保文件在不同版本的 Photoshop 中打开时保持一致。默认为“询问”。

--总不

Never 

保存 PSD 或者 PSB 时不考虑兼容旧版本的 Photoshop。这样的话，新版本保存的 PSD 或者 PSB 就有可能在旧版本的 Photoshop 中无法打开。 

--总是

Always 

保存 PSD 或者 PSB 时总是兼容旧版本的 Photoshop。

--询问

Ask

保存 PSD 或者 PSB 时将弹出对话框，可自行选择是否“最大兼容”。

近期文件列表包含 X 个文件

Recent File List Contains X Files

设置在“文件”菜单中显示的“最近打开文件” Open Recent的数量，默认为 20 个，最多可达 100 个。
