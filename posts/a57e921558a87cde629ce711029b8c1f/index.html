Python-OpenCv中的cv2.VideoCapture()类

OpenCv中提供了VideoCapture类的构造方法VideoCapture()

cv2.VideoCapture()具有两个功能，一是可以完成摄像头的初始化，打开摄像头；二是可以完成视频文件的初始化。

摄像头的初始化：

capture = cv2.VideoCapture(index)

参数说明：

        capture：要打开的摄像头

        index：摄像头的设备索引，对于笔记本电脑来说，为0时，一般打开的是电脑的内置摄像头，为1时，打开的为外接的摄像头。

下面是一段打开摄像头并展示画面的代码示例：

import cv2

capture = cv2.VideoCapture(0)   # 打开笔记本内置摄像头
while capture.isOpened():      # 笔记本摄像头被打开
    retval, image = capture.read()  # 从摄像头中实时读取画面
    cv2.imshow("Video", image)  # 在窗口显示读取到的视频
    key = cv2.waitKey(1)     # 等待用户按下键盘按键的时间为1毫秒
    if key == 32:        # 如果按键为空格，就跳出循环
        break
capture.release()   # 关闭摄像头
cv2.destroyAllWindows()    # 销毁窗口

视频文件的初始化：

video = cv2.VideoCapture(filename)

参数说明:

        video：要打开的视频

        filename：打开的视频文件名，可以为相对路径或绝对路径（在我电脑上，这里文件名为中文也是可以正常运行的），可以支持各种格式的视频文件，例如：MP4、AVI、MOV、WMV、FLV等后缀名，在不同操作系统中所支持的视频文件格式不同。但是不同操作系统都支持以“.avi”为后缀的视频文件

下面是一段播放本地视频的代码示例：

import cv2 as cv

video = cv.VideoCapture("gs.mp4")  # 打开视频文件
while video.isOpened():
    ret, img = video.read()     # 视频文件被打开
    # 设置“Video”窗口的长宽属性为420，300
    cv.namedWindow("Video", 0)
    cv.resizeWindow("Video", 420, 300)
    if ret:             # 读取到视频文件
        cv.imshow("Video", img)     # 在窗口显示读取到的视频文件
    else:
        break
    key = cv.waitKey(1)
    if key == 27:       # 如果按键为ESC
        break
video.release()             # 关闭视频文件
cv.destroyAllWindows()

isOpened()

retval = cv2.Videocapture.isOpened()

参数说明：retval：isOpened()方法的返回值。摄像头初始化成功就返回True，否则返回False

在VideoCapture()语法格式的基础上，isOpened()方法可以简写，如下：

cap = cv2.VideoCapture(0)
retavl = capture.isOpened()

read()

摄像头初始化之后，要想从摄像头中读取帧，就要用到read()方法

retval, image = cv2.VideoCapture.read()

参数说明：retval：是否读取到帧，读取到为True，否则为False

                  image：读取到的帧（画面）。视频是由一帧一帧的图像构成的，所以image取到的只                    是一张图片

对此方法同样可以进行简写，如下：

capture = cv2.VideoCapture(0)
retval, image = capture.read()

release()

在不用摄像头的时候要关闭摄像头，release()用来关闭摄像头

cv.VideoCature.release()

简写为：

capture = cv2.VideoCapture(0)
capture.release()

案例

打开摄像头并实时显示画面

import cv2

capture = cv2.VideoCapture(0)   # 打开笔记本内置摄像头
while capture.isOpened():      # 笔记本摄像头被打开
    retval, image = capture.read()  # 从摄像头中实时读取视频
    if capture.isOpened():
        cv2.imshow("Video", image)  # 在窗口显示读取到的视频
    key = cv2.waitKey(1)     # 等待用户按下键盘按键的时间为1毫秒
    if key == 32:
        break
capture.release()   # 关闭摄像头
cv2.destroyAllWindows()
