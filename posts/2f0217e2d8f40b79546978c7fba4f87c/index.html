在 Ubuntu 20.04 安装 Redis


1. 先切换到 root 用户

在 Ubuntu 20.04 中，可以通过以下步骤切换到 root 用户：

  1. 输入以下命令，以 root 用户身份登录：
sudo su -
  1. 按回车键，并输入当前用户的密码（即具有 sudo 权限的用户的密码）
  2. 如果密码正确，将会切换到 root 用户，并且提示符会变为以 root@yourhostname 格式显示

    注：这里 root@后面的是一串字符串，与博主使用的是阿里云的服务器有关


2. 使用 apt 命令来搜索 redis 相关的软件包

apt search redis




3. 使用 apt 命令安装 redis

apt install redis




4. 输入命令检验是否安装成功

netstat -anp | grep redis

出现以下界面，则说明安装成功

当前状况：由于当前端口对应的 ip 127.0.0.1 是环回 IP，只能由当前主机上的客户端访问，跨主机就访问不了
实际需求是其它主机也能够访问到，所以需要修改当前端口对应的 ip
