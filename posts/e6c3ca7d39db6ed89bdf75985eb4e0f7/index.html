idea/webstorm配置vue项目启动报Please specify npm scripts to run错误

配置vue项目不识别scripts，原因没识别到项目中的package.json文件内容

解决方法很简单：
File->Settings->Editor->File Types
目录下翻找到JSON，在File name patterns中添加*.json

配置完回来看scripts已经出现可选项
