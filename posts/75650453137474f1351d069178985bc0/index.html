JDK 22 详细安装教程

JDK官方下载网址：Java Downloads | Oracle

步骤 1: 下载 JDK 22

  1. 打开你的浏览器，访问 Oracle 官方网站的 JDK 下载页面。

  2. 在 “Java SE 22” 部分，选择适用于 Windows 64 位操作系统的版本。

  3. 点击“下载”按钮，接受许可协议并下载安装程序（通常是一个 .exe 文件）。

步骤 2: 运行安装程序

  1. 双击下载的 .exe 文件，启动安装程序。

  2. 在安装向导中，点击“下一步”继续。

步骤 3: 设置安装选项

  1. 在安装类型选择窗口中，选择“自定义”选项，然后点击“下一步”。

  2. 在自定义安装窗口中，可以选择安装的组件。确保至少选择了 “Development Tools” 和 “Source Code”。点击“下一步”。

步骤 4: 选择安装位置

  1. 在安装位置窗口中，可以选择安装 JDK 的位置。默认情况下，它会安装在 C:\Program Files\Java\jdk-22。你也可以点击“更改”按钮来指定其他位置。点击“下一步”。

步骤 5: 完成安装

  1. 点击“安装”按钮，开始安装 JDK。

  2. 安装过程可能需要一段时间，请耐心等待。

  3. 安装完成后，点击“完成”按钮退出安装程序。

步骤 6: 配置系统环境变量

  1. 右键点击桌面左下角的“开始”按钮，选择“系统”。

  2. 在系统窗口中，点击“关于”选项卡。

  3. 点击“系统信息”。

  4. 在系统信息窗口中，点击“高级系统设置”链接。

  5. 在系统属性窗口中，点击“环境变量”按钮。

  6. 在系统变量部分，找到名为 PATH 的变量，双击它以编辑。

  7. 在变量值字段的末尾，添加 JDK 的安装路径。例如，如果 JDK 安装在 C:\Program Files\Java\jdk-22，则添加 ;C:\Program Files\Java\jdk-22\bin 到变量值的末尾（请确保前后以分号分隔）。

  8. 点击“确定”关闭所有打开的窗口。

步骤 7: 验证安装

  1. 打开命令提示符（按下 Win + R，键入 cmd，然后按 Enter）。

  2. 在命令提示符中，键入 java -version 并按 Enter。

  3. 如果安装成功，你应该会看到 JDK 的版本信息。
