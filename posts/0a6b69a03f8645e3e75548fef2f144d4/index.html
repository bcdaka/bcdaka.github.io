解决python调用matplotlib绘制图像只弹出空白无响应窗

在调用matplotlib绘制图像时，弹出绘制窗口，但没有响应并且很快程序结束

查找资料发现两种解决方案：

1.修改导入包代码

import matplotlib
matplotlib.use('Qt5Agg')
import matplotlib.pyplot as plt

2.修改显示代码

plt.show(block = True)

以上两种方法均没有奏效，考虑python版本与matplotlib版本不适配

解决方案：

1.创建新环境python=3.10

2.下载相应版本安装包

成功解决
