四、疑难杂症软件 -- 卸载方法

1.在系统的运行中输入cmd打开cmd命令窗口；

 2.在命令窗口中输入WMIC然后回车；

 3.然后输入Product list等待一段时间(大约5分钟)后就会列出

当前系统上安装的应用程序Microsoft SQL Server XX RsFx Driver； 

 

   4.找到需要写在的程序将其 IdentifyingNumber号码复制下来；

  5.然后输入exit；

  6.最后输入msiexec  /x{复制的号码}命令行进行卸载。
