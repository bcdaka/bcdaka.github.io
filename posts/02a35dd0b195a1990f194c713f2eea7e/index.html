dm8修改sysdba用户的密码

1 查看达梦数据库版本

SQL> select * from v$version;

LINEID     BANNER                           
---------- ---------------------------------
1          DM Database Server 64 V8
2          DB Version: 0x7000c
3          03134283904-20220630-163817-20005

2 查看参数设置

SQL> select name,value,sys_value,file_value from v$parameter where name like '%OSAUTH%';

LINEID     NAME                 VALUE SYS_VALUE FILE_VALUE
---------- -------------------- ----- --------- ----------
1          ENABLE_REMOTE_OSAUTH 0     0         0
2          ENABLE_LOCAL_OSAUTH  0     0         0

3 修改参数，并重启数据库实例


修改： 

SQL> alter system set 'ENABLE_LOCAL_OSAUTH'=1 spfile;
DMSQL executed successfully
used time: 1.151(ms). Execute id is 1203.
SQL> select name,value,sys_value,file_value from v$parameter where name like '%OSAUTH%';

LINEID     NAME                 VALUE SYS_VALUE FILE_VALUE
---------- -------------------- ----- --------- ----------
1          ENABLE_REMOTE_OSAUTH 0     0         0
2          ENABLE_LOCAL_OSAUTH  0     0         1

used time: 3.097(ms). Execute id is 1204.

重启数据库实例：

[root@test1 ~]# systemctl restart DmServiceDBSERVER.service

SQL> select name,value,sys_value,file_value from v$parameter where name like '%OSAUTH%';

LINEID     NAME                 VALUE SYS_VALUE FILE_VALUE
---------- -------------------- ----- --------- ----------
1          ENABLE_REMOTE_OSAUTH 0     0         0
2          ENABLE_LOCAL_OSAUTH  1     1         1

4 sysdba用户可以修改密码了

SQL> conn sysdba/Dameng123

服务器[LOCALHOST:5236]:处于普通打开状态
登录使用时间 : 30.012(ms)
SQL> 
SQL> alter user sysdba identified by Oracle123;
操作已执行
已用时间: 108.525(毫秒). 执行号:700.
SQL> 
SQL> exit
[dmdba@test ~]$ disql 
disql V8
SQL> conn sysdba/Dameng123
[-2501]:用户名或密码错误.
SQL> conn sysdba/Oracle123

服务器[LOCALHOST:5236]:处于普通打开状态
登录使用时间 : 2.914(ms)
