Mac安装telnet

一、安装Homebrew

1、打开官网：Homebrew — The Missing Package Manager for macOS (or Linux)
2、打开终端输入：
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

二、安装Telnet

brew install telnet

三、执行Telnet命令

telnet api.evcoming.com 5131
