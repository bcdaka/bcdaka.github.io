ESP32 IDF ADF 加入音频

需要把mp3制作成音频bin

用ADF自带工具

果用户需要生成自己的 audio-esp.bin，则需要执行 mk_audio_bin.py 脚本（位于 $ADF_PATH/tools/audio_tone/mk_audio_tone.py），并且指定相关文件的路径。

源 MP3 文件在 tone_mp3_folder 文件夹中，生成的 C 文件、H 文件以及二进制 bin 文件都存放在此目录下。

python3 $ADF_PATH/tools/audio_tone/mk_audio_tone.py -f ./ -r tone_mp3_folder

mp3文件放到mytone_mp3_folder
执行

  python mk_audio_tone.py -f ./ -r mytone_mp3_folder

将后面两个生成的资源文件也件加入
