拉Mysql镜像：

docker pull mysql:latest

查看镜像：

docker images

创建mysql数据卷目录(记得查看)：

mkdir -p /data/mysql

构建Mysql容器：

docker run -d \
  --name mysql \
  -v /data/mysql:/var/lib/mysql \
  -e MYSQL_ROOT_PASSWORD=123456 \
  -p 3306:3306 \
  mysql:latest
