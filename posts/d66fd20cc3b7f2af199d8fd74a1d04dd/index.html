Mac 彻底删除 node 和 npm

1、通过 homebrew 安装的

通过 node -v 查看版本好

然后使用指令删除： 

brew uninstall node@版本号 --force

例：安装的是 v16.20.0，使用brew uninstall node@16 --force 进行删除；

2、通过官网 pkg 安装包下载的

直接使用：

sudo rm -rf /usr/local/{bin/{node,npm},lib/node_modules/npm,lib/node,share/man/*/node.*}
