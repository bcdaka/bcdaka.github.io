Linux系统下虚拟机中安装（搭建）mysql的详细教程

1、启动虚拟机

①打开VMware、MobaXterm，启动虚拟机集群；

②上传mysql的rpm安装包至linux系统下目录（只需有一台安装，我们在第一台虚拟机上安装）；

2、检查mysql的安装环境

①输入代码，检查是否存在 mariadb：

rpm -qa | grep mariadb

②如图所示查询到mariadb，需要先进行卸载：

rpm -e --nodeps mariadb-libs

③再次输入命令 rpm -qa | grep mariadb 进行检查，没有出现mariadb即为ok；

3、安装mysql的rpm包

①进入mysql压缩包所在目录:

 cd /opt/install_packages/

②解压到当前目录：

tar -xf mysql-5.7.28-1.el7.x86_64.rpm-bundle.tar

③第一步安装common，如图所示即为成功：

rpm -ivh /opt/install_package/mysql-community-common-5.7.28-1.el7.x86_64.rpm

  第二步安装libs：

rpm -ivh /opt/install_package/mysql-community-libs-5.7.28-1.el7.x86_64.rpm --nodeps --force

  第三步安装libs-compat：

rpm -ivh /opt/install_package/mysql-community-libs-compat-5.7.28-1.el7.x86_64.rpm

  第四步安装client：

rpm -ivh /opt/install_package/mysql-community-client-5.7.28-1.el7.x86_64.rpm

  第五步安装server：  

rpm -ivh /opt/install_package/mysql-community-server-5.7.28-1.el7.x86_64.rpm --nodeps --force

④安装完成，删除rpm包：   rm -rf *.rpm 

4、mysql已安装完成，为其更改配置

第一步、删除/var/lib/mysql 目录下所有内容

①进入配置文件目录：   cd /etc/ 

②编辑配置文件： vim my.cnf

③复制如图所示目录： /var/lib/mysql       输入：q退出

④进入所复制的目录下，删除里面所有内容：

cd /var/lib/mysql
rm -rf ./*

第二步、修改mysql密码

①初始化mysql数据库

mysqld --initialize --user=mysql

 ②查看root用户的临时密码：

cat /var/log/mysqld.log

③复制保存好临时密码：   2%;vavrhoZi4

④启动mysql：  

systemctl start mysqld

  还可查看mysql的状态，running则为运行成功：

systemctl status mysqld

⑤登录mysql时，需输入密码（输入的密码不会显示出来，记得只需粘贴一次密码即可），如图：

mysql -uroot -p

⑥修改mysql密码，输入 exit; 退出；

set password = password('root123');

⑦输入命令  mysql -uroot -p    ，输入新密码重新登录mysql；

第三步、修改root用户的权限

①修改root用户的iP访问限制，允许任意ip都可以访问

update mysql.user set host='%' where user='root';

报错语句结构错误，如图所示：

更改如下，重新输入代码，更新成功：

UPDATE mysql.user 
SET host = '%' 
WHERE user = 'root';

②查看是否修改成功，如图所示：

SELECT host FROM mysql.user
WHERE user='root';

③刷新权限：   flush privileges;

 报与上个同样的错，如图所示：

更改：手写即可成功， 输入 exit;  退出。
