Docker 安装消息队列RabbitMQ

拉取镜像

docker pull rabbitmq

拉取最新镜像

创建并运行

 docker run -d 
 --hostname my-rabbit 
 --name rabbit
  -p 15672:15672 
  -p 5673:5672
   rabbitmq

开启Web管理

进入容器

docker exec -it rabbitmq /bin/bash

开启web管理

rabbitmq-plugins enable rabbitmq_management

http://localhost:15672 进入管理端
账号密码默认都是guest

各端口作用

  • AMQP: 5672 - 默认的消息传递协议端口客户端连接。

  • 管理控制台: 15672 - Web 界面用于管理和监控。

  • AMQP（SSL/TLS）: 5671 - 加密的 AMQP 连接端口。
