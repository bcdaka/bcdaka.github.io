循环结构是必须要学习好的，几乎所有代码都会应用到循环结构

一、while循环

while语句书写格式如下：

while (表达式)
{
     语句；
}

在使用while循环时，确保循环前已完成了初始化，并在循环体内适当地调整循环控制变量。

例题：使用while循环执行n行hello打印

int main()
{
	int n = 0;
	scanf("%d", &n);
	while (n>0)
	{
		printf("hello\n");
		n--;
	}
	return 0;
}

代码意思：输入一个变量，如果输入的n大于0那么就执行while语句，直到n>0才执行，每循环一次函数，n就减一，因为 n-- 在 printf 函数的后面，所以无论是先减 n-- 还是后减，循环都能正常结束。”

二、for循环

for循环是使用最多的循环函数

for循环也是很简单的，语法如下：

for(表达式1; 表达式2; 表达式3)
{
    语句；
}

写for循环体我们必须写三个条件：

表达式1、初始化 。

表达式2、⽤于循环结束条件的判断。

表达式3、⽤于循环变量的调整

例题：使用for循环打印1-10。

int main()
{
	int i = 0;
	for ( i = 1; i <=10; i++)
	{
		printf("%d ", i);
	}
	return 0;
}

代码意思：设置初始值i=1 ， 当i<=10才终止程序，每次循环语句每次执行一次i++一次，此时就可以成功打印出1-10个数字。

三、do while循环

do while循环语句如下：

do
{
	语句
} while (表达式);

do while循环这个函数，它是先执行命令，如何在判断判断，这是跟for和while循环的显著的区别

例题：打印1-10数字

int main()
{
	int i = 1;
	do
	{
		printf("%d ", i);
		i++;
	} while (i<=10);
	return 0;
}

对比for跟while循环可以看出，先执行printf语句，然后再执行其他的。

一般do while应用于至少执行一次的场景，这还是我们所说的，不管三七二十一，先直接语句

四：break、continue、goto语句

一、break语句

break语句的用法在于，我们我们执行完一个循环语句以后，不想执行接下来的语句了，我们就可以使用break来终止程序

例题：打印1-10个数字，在打印到5的时候，终止程序。使用while循环（使用for跟do while循环也是可以的）

int main()
{
	int i = 1;
	while (i<=10)
	{
		printf("%d ", i);
		if (i == 5)
		{
			break;
		}
		i++;
	}
	return 0;
}

我们可以看到，在if语句中，我们设置的条件是当i等于5时，就执行break语句，最终在i=5时最后程序终止。

二、continue语句

continue语句是用于跳过某个特定的条件，直到最终函数执行完毕。

例题：打印1-10，不打印5。三个循环函数可以任意选择实现，我们这次选择for循环

int main()
{
	int i = 0;
	for ( i = 1; i <= 10; i++)
	{
		if (i == 5)
		{
			continue;
		}
		printf("%d ", i);
	}
	return 0;
}

我们可以看到，最终没有打印5这个数字。

三、goto语句

goto语句可以实现函数在同一函数内跳转

例题如下：

int main()
{
	goto next;
	printf("我不会被执行");
next:
	printf("我跳到这边了");
	return 0;
}

goto 语句之后的 printf 将不会被执行，因为 goto 会立即跳转到标签 next:。

虽然goto在现代编程实践中较少使用，因为它可能导致代码难以理解和维护，但在某些特定场景下，如跳出多层嵌套循环或长跳转，它仍然有其用途。
