SQL Server使用教程_初学者必备

一、编写的目的

        很多刚接触 SQL Server 的人，在安装完后都会有一个疑问，我怎么去使用 SQL Server 呢？

这篇文章就是为初学者量身定制的，让你从初学到入门，分分钟掌握 SQL Server。

二、启动SQL server服务的常用方法

        1.方法一：使用cmd命令启动 SQL server服务

步骤：

        （1）使用Windows搜索功能，输入cmd,如下图所示：

        

        （2）选择命令提示符，鼠标右击，选择“以管理员身份运行”（注意：一定要“以管理员身份运行”，否则启动SQL Server服务会失败！）如下图：

        

        （3）输入以下的代码：

net start mssqlserver

输入代码之后按回车键，会自动启动 SQL Server 服务，等待几秒后提示“服务已经启动成功”如下图所示：

        此时显示 SQL Server服务已经启动成功，SQL Server服务启动之后，可以打开 Microsoft SQL Server Management Studio 正常使用了。

第三章节会介绍在哪里打开 SQL Server Management Studio 。

 说明：        

        如果启动时，遇到发生系统错误，拒绝访问的提示，原因可能是因为没有“以管理员身份运行”命令提示符，重新执行（2）和（3）步骤。

        2.方法二：使用 SQL Server 配置管理器启动 SQL Server服务

步骤：

        （1）使用Windows搜索功能，输入SQL Server，如下图所示：

        

        （2）输入后，会有SQL Server 配置管理器或SQL Server XXX 配置管理器显示（SQL Server 版本不同，显示也不同，只要是配置管理器就行） ，选中SQL Server 配置管理器“以管理员身份运行”，如下图：

       

        （3）点击“以管理员身份运行”后，会打开 SQL Server 配置管理器 窗口，如下图：

       

        （4）左侧选择“SQL Server服务”（进去默认选择该选项），右侧选择需要启动的服务，右击选择“启动”（刚接触 SQL Server ，可以启动 SQL Server服务下的所有服务，无法启动的服务可以忽略掉不用管） ，如下图所示：

        此时显示 SQL server服务已经启动成功，SQL Server服务启动之后，可以打开 Microsoft SQL Server Management Studio 正常使用了。

三、数据库的基础操作

        上面步骤已经启动 SQL Server服务 了，接下来我们就可以对数据库进行操作啦。

1.打开Microsoft SQL Server Management Studio

步骤：

        （1）开始→找到“Microsoft SQL Server XXX→选择“SQL Server Management Studio”

       

        （2）打开 SQL Server Management Studio 后窗口如下图：

        服务器类型：默认选择就行‘’

        服务器名称：默认选择是本地服务器，或者自己输入自己电脑的IP；

        身 份 验 证：默认选择“Windows身份验证”，也可以选择“SQL Server”身份验证；（这个看安装的时候是否是选择“混合模式”）

        （3）选择登录模式后，点击“连接”按钮，将连接成功，如下图：

2.新建数据库

        因为本文面向刚接触 SQL Server 的初学者，所以本文将不讲述用 SQL语句对数据库操作的方法。

步骤：

        （1）选中“对象资源管理”中的数据库，鼠标右击“数据库” ,点击“新建数据库”，弹出的新建窗口中，输入数据库名称（中英文都可以），初始大小根据个人需要设置，储存路径默认就好。设置完后点击右下角的“确定”按钮即可。如下图所示：

        

        （2）在“对象资源管理器”中，点击“数据库”的旁边的“加号”，点击这个“【+】”，即可找到我们刚刚新建的数据库，如下图所示如下图所示：

       

         点击旁边的“+”，就可以看到该数据库下的详细目录 ，如下图所示

 然后我们可以通过SQL语句检查我们的数据库是否建立成功，点击“新建查询”，输入SQL语句后执行，SQL语句如下：

select * from sys.databases

执行后，会返回当前所有的数据库，我们可以看到已经返回了我们刚刚新建的数据库，其它为系统数据库。如下图所示：

注意：建议不要对系统数据库操作。

新建表步骤：

        （1）在“对象资源管理器”中，选中表，鼠标右击表，选择“新建表”，如下图所示：

接下来我们需要设置列名，数据类型，是否允许Null值（null表示为空）

显然列名就是我们学生的

编号

性别

数据类型这里只介绍常用的:

tinyint       小整型数据，一般是1,2,3这类的数据；

bigint        大整型数据，一般表示比较大的数据类型；

float(n)      可选精度的浮点数（实数），精度至少为n为数字；

char(n）    长度为n的定长字符串 varchar(n) 最大长度为n的变长字符串；

varchar(n) 最大长度为n的变长字符串。

        由于学生人数较多，编号的数值可能较大，所以用 bigint；学生姓名名、性别显然是字符串，我们可以用char或者varchar，这里选择用varchar（因为varchar可以节省存储空间，虽然节省的这点存储空间没有用），根据上述的分析，我们的数据类型的选择也已经出来了。

        是否允许null值，也就是是否允许空值的出现，根据具体情况来判断，比如“学号”和“姓名”，这些值必须存在，不然就没有意义了，可以设置不允许空值的出现。对于性别和年龄可有可无，影响不大，所以设置允许空值，如果设置了允许空值，在插入数据的时候，遇到空值，用NULL表示。

        

        （2）保存新建的表，使用快捷键 ctrl+s 保存，或按下图步骤保存，如下图所示：

点击保存后弹出填写 表名称窗口，根据自己需求，填写好表名称，点击“确定”即可。

在“对象资源管理器”中，我们可以查看到新建的表数据，如下图所示：

       

        （3）往新建表中添加数据，选择我们新建的表，右键后选择“编辑前200行”，如下图所示：

依次插入数据（注意：在插入数据的时候，建议一行一行的插入），如果遇到感叹号，可以不用管，继续插入数据的操作，插入完之后，结果如下图所示：
