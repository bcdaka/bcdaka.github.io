删除node_modules文件夹后执行命令

npm update

仍然报错缺少“@vue/cli-plugin-babel”

在网上找了半天的方法，才找到只言片语。

原因：当前电脑nodejs版本过高，项目的配置太低，不匹配，如下package.json

{

  "dependencies": {
    "node-sass": "^4.13.1",
    "sass-loader": "^7.3.1",
  },
  "devDependencies": {
    "@vue/cli-plugin-babel": "~4.5.15",
  },
  
}

node 版本与 node-sass 版本的对应关系：

查看文档的地址： https://www.npmjs.com/package/node-sass

解决方法，删除这几行配置，重新安装。

npm install node-sass
npm install sass-loader
npm install @vue/cli-plugin-babel

重新运行问题解决

npm run serve
