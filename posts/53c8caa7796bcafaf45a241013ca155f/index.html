算法基础——排队打水（贪心算法）

AcWing 913. 排队打水
有n个人排队到1个水龙头打水，他们装满水的时间为t1，t2，t3，…tn，为整数且各不相等，应如何安排他们的打水顺序才能使所有人的等待时间之和最小？求出最小的等待时间之和？

典型的贪心问题，根据我们的直觉判断就是，让装满水时间越短的人排在越靠前面的位置，那么所有人等待的时间之和就会最短!(因为越排前面打水的人，后面等待的人数就越多，即有更多的人需要等待【排前面打水人所用的时间t】，导致等待时间之和增加)。

证明猜想：
假设 t [ i ] > t [ i + 1 ] t[i] > t[i+1] t[i]>t[i+1]

按照我们的贪心策略计算等待时间之和： t 贪 = t [ 1 ] ⋅ ( n − 1 ) . . . + t [ i ] ⋅ ( n − i ) + t [ i + 1 ] ⋅ ( n − i − 1 ) + . . . + t [ n − 1 ] ⋅ 1 + t [ n ] ⋅ 0 t_贪=t[1]\cdot(n-1)...+t[i] \cdot (n-i) + t[i+1] \cdot (n-i-1)+...+t[n-1]\cdot1+t[n]\cdot0 t贪​=t[1]⋅(n−1)...+t[i]⋅(n−i)+t[i+1]⋅(n−i−1)+...+t[n−1]⋅1+t[n]⋅0

交换任意的i和i+1的位置后等待时间之和： t 换 = t [ 1 ] ⋅ ( n − 1 ) . . . t [ i + 1 ] ⋅ ( n − i ) + t [ i ] ⋅ ( n − i − 1 ) + . . . + t [ n − 1 ] ⋅ 1 + t [ n ] ⋅ 0 t_换=t[1]\cdot(n-1)...t[i+1] \cdot (n-i) + t[i] \cdot (n-i-1)+...+t[n-1]\cdot1+t[n]\cdot0 t换​=t[1]⋅(n−1)...t[i+1]⋅(n−i)+t[i]⋅(n−i−1)+...+t[n−1]⋅1+t[n]⋅0

由于 t [ i ] > t [ i + 1 ] t[i] > t[i+1] t[i]>t[i+1],可以得出 t 贪 − t 换 < 0 , 即 t 贪 < t 换 t_贪-t_换<0,即t_贪<t_换 t贪​−t换​<0,即t贪​<t换​。

根据该贪心策略，得出的最容易想到的做法AC代码如下：

#include<iostream>
#include<algorithm>
using namespace std;
const int N = 100010;
int t[N];
int main() {
	ios::sync_with_stdio(false);
	cin.tie(nullptr);
	int n;
	cin >> n;
	//输入每个人打满水的时间
	for (int i = 0; i < n; i++) cin >> t[i];
	//将打满水的时间升序排序
	sort(t, t + n);
	long long res = 0;
	for (int i = 0; i < n; i++) {
		//根据制定的贪心策略，依次计算某个人正在打水的人，后续x个人等待的时间和
		//所有等待时间和就是总的等待时间和
		res += t[i] * (n - i - 1);
	}
	cout << res << endl;
	return 0;
}

利用前缀和思路解题，如下图所示，图中已经默认1，2，3，4，5…分别按时间升序排序即t1<t2<t3<t4<t5…。此处，我们通过求出前缀和数组s[1],s[2],s[3],s[4],s[5]…来求所有等待时间之和。不难看出，某个人的等待接水的时间之和就是他前一个的前缀和数组(即4号的排队的等待时间就是s[3],5号的排队时间就是s[4]),因此，总的所有人的等待时间之和就是s[0]+s[1]+s[2]+s[3]+s[4]。

由此可得，AC代码为：

#include <iostream>
#include <cstring>
#include <algorithm>

using namespace std;

const int N = 100010;
int a[N];

int main()
{
    int n;
    cin >> n;
    for(int i = 1; i <= n; i++) cin >> a[i];
    //按接满水时间升序排序
    sort(a + 1, a + n + 1);
    //求前缀和数组
    for(int i=1;i<=n;i++)
        //此处原本应该s[i]=s[i-1]+a[i]
        //为了节省空间，直接用原数组a[i]来存储前缀和数组
        a[i]+=a[i-1];
    
    //求所有人的总等待时间
    long long res=0;
    //此处i=0或1都可以
    //为了方便理解，直接i=0，即s[0]为第一个人的等待时间
    for(int i=0;i<n;i++)
        res+=a[i];

    cout<<res<<endl;

    return 0;
}

排队打水的进一阶
如果此处，我们将原来的题目更改以下，将水龙头变为不止1一个呢？
有n个人排队到m个水龙头打水，他们装满水的时间为t1，t2，t3，…tn，为整数且各不相等，应如何安排他们的打水顺序才能使所有人的等待时间之和最小？求出最小的等待时间之和？

此时，我们仍然可以按照贪心策略的思路来做。如下图所示，对比单个水龙头的前缀和思路思考。多个水龙头进行排队，其实我们仍然只需要按照贪心策略思路来进行。
1、将接满水的时间按照升序排序
2、将接满水时间越短的，越往前面排。由于是多个水龙头，因此根据接满水的时间长短，先横向把水龙头占满，再纵向排队。
3、求出每个水龙头中，对应等待时间的纵向前缀和数组，如 s [ 1 ] , s [ r + 1 ] , s [ 2 r + 1 ] . . . s[1],s[r+1],s[2r+1]... s[1],s[r+1],s[2r+1]...。
4、求出每个人的等待时间和，以求接满水时间排第 i i i的人等待时间和为例子， t 等 = s [ i − r ] t_等=s[i-r] t等​=s[i−r]。
5、求出所有人等待时间总和。将所有 t 等 t_等 t等​加和起来，得到res。


AC代码：

#include<iostream>
#include<algorithm>
using namespace std;

const int N = 100010;

int res, a[N], s[N];


int main() {
    int n, r;
    cin >> n >> r;
    for (int i = 1; i <= n; i++)
        cin >> a[i];
    sort(a + 1, a + n + 1);
    //前r个前缀和直接等于本身
    for (int i = 1; i <= r; i++) s[i] = a[i];

    //求纵向前缀和
    for (int i = r + 1; i <= n; i++) s[i] = s[i - r] + a[i];

    //求总等待时间
    for (int i = 1; i <= n-r; i++) res += s[i];

    cout << res << endl;
    return 0;
}
