VSCode创建Vue 项目

  1. 下载node.js 安装；
    安装后环境变量添加，path。
  2. node -v
    npm -v
    npm install -g @vue/cli

vue --version

–
vue create test_vue
手动配置 test_vue_config (应该有默认的，不用手动配置)

– 启动项目
cd test_vue
npm run serve
http://localhost:8080/

–安装axios
npm install axios
