Linux系统安装opencv

在Linux系统上安装OpenCV时，使用pip install opencv-python-headless是一个常见的选择。以下是安装OpenCV的步骤总结：

安装步骤：

  1. 更新系统软件包（可选）：

    sudo apt update
sudo apt upgrade

  2. 安装pip（如果尚未安装）：

    sudo apt install python3-pip

  3. 安装OpenCV：
    使用以下命令安装OpenCV的无GUI版本（适用于服务器或不需要图形界面的环境）：

    pip install opencv-python-headless

说明：

  • opencv-python：包含完整的OpenCV库，包括GUI功能（需要X11等图形环境）。
  • opencv-python-headless：去除了GUI功能，适合在没有图形界面的服务器上使用，减少依赖和安装包大小。

验证安装：

可以通过以下Python代码来验证是否成功安装：

import cv2
print(cv2.__version__)

如果能够正确输出OpenCV的版本号，则表示安装成功。
