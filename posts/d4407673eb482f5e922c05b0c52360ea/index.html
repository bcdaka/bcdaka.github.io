        以下就为大家介绍几款不错的 Redis 可视化工具，本人平时常用：Redis DeskTop Manager 和 Another Redis Desktop Manager github 两款都非常不错。

一、Redis DeskTop Manager  桌面端 Redis 可视化工具

        Redis DeskTop Manager 工具应该是现在使用率最广的可视化工具了，也不用多说什么了。运营时间很久。经过了数次迭代。跨平台支持。现在虽然收费了。但是也不贵，一年才百来块，而且旧版本是免费的，界面看着比较简洁，功能很全。除了基本的五大数据类型之外，还支持Stream数据类型，在value的显示方面。基本支持所有数据类型的显示。

  

二、Another Redis Desktop Manager github 桌面端 Redis 可视化工具 

        Another Redis Desktop Manager github 怎么说明，只能用小巧玲珑来说了，页面非常简洁美观，基本的功能都有，支持多平台、监控统计、页面主题、集群、搜索，分组显示，多选操作。整体感觉比较不错。 最主要的是简洁免费，有想法的可以动身了， 这软件是国内一个大佬嫌弃各种可视化工具都不顺手，所以自己开发的。

         Another Redis Desktop Manager  Github下载地址：https://github.com/qishibo/AnotherRedisDesktopManager/releases

        Another Redis Desktop Manager  Gitee下载地址：https://gitee.com/qishibo/AnotherRedisDesktopManager/releases

三、 Redis Insight  WEB版 Redis 可视化工具

        这个是收费的，但是不算贵，个人版一年200多R币对于他的功能来说，性价比很高了，Redis Insight 是官方推出的一款工具，他的强大可以说爆了其他可视化工具几条街，目前2.38版本我个人感觉不太好用，有很多功能含糊不清，可能是我没琢磨明白，但是齐一些列功能是实打实的，全方面都很不错，所以我放在最后一个推荐。

        Redis Insight 官方下载地址：https://redis.com/redis-enterprise/redis-insight/#insight-form
