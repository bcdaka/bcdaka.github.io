a+=1 和a = a+1的区别

一、实例代码

public class Test {
	public static void main(String[] args) {
		byte a = 10;
//		a = a + 1;  
//		a = (byte) (a + 1);
		a += 1;
		System.out.println(a);
	}
}

上面的对变量a进行加一操作时，使用了不同的方法

二、代码解释

对于注释的两行，第一行是错误的方法，因为在java中进行运算的时候默认使用的是int类型，并且在java中高精度向低精度转化的时候需要声明，因此第二行注释是正确的。

对于+=而言，不用注重类型转化

三、总结

综上所述，+=不用管类型转化，=需要注重类型转化
