常用浏览器禁用JavaScript方法

在安全测时有一些验证逻辑是在前端进行验证的，我们就可以通过浏览器禁用JavaScript来绕过这些验证，下面是几款常用浏览器禁用JavaScript的方法

  • Edge

    1. 打开 Edge 浏览器，并在地址栏中输入 about:flags ，然后按 Enter 键，进入 Edge 的实验性功能页面。

    2. 在实验性功能页面中，找到 “JavaScript” 选项，并选择 “Disabled” 选项。

    3. 关闭 Edge 浏览器，并重新打开它。

    4. 在 Edge 浏览器中，单击地址栏右侧的三个点按钮，然后选择 “设置” 选项。

    5. 在设置页面中，选择 “Privacy, search, and services” 选项。

    6. 在隐私设置页面中，找到 “Cookies and site permissions” 选项，并单击它。

    7. 在 “Cookies and site permissions” 页面中，找到 “JavaScript” 选项，并选择 “Block” 选项。

    8. 禁用 JavaScript 后，您可以重新加载网页，JavaScript 将不再运行。

  • Chrome

    1. 地址栏输入： chrome://settings/content/javascript
    2. 然后找到JavaScript选择进行禁用即可

  • Firefox

    1. 地址栏输入：about:config
    2. 找到 javascript.enabled 将其转换为false

  • IE

    1. 先打开IE浏览器，点击右上角的 工具
    2. 点击 Internet选项
    3. 点击 安全 选项卡
    4. 在 安全 选项卡里找到 自定义级别 ，点击 自定义级别
    5. 选择禁用 Java小程序脚本 和 活动脚本
    6. 弹出警告框点击 是
    7. 然后重启浏览器

  • Opera

    1. 单击 Opera 菜单。在“设置”选项上方悬停鼠标。如找不到 Opera 菜单，只看到传统菜单栏，则单击“工具”。

    2. 在“快速偏好”上方悬停鼠标。勾选名为“启用 JavaScript”的复选框。

    3. 刷新当前网页。这将载入您的新设置。
