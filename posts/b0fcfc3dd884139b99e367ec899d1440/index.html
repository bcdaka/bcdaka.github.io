解决： phpstudy 无法启动 MySQL

一、问题

二、解决方法分类

  1. 电脑没有安装 MySQL
  2. 电脑安装了 MySQL

三、电脑没有安装 MySQL


若还是不行，找到安装的 phpstudy 目录 ---- Extensions ---- MySQL 文件夹中的 my.ini 查看其中的port、basedir、datadir 是否正确

四、电脑安装了MySQL

  1. 在桌面找到控制面板 — 点击 — 在搜索栏中搜索服务 — 找到查看本地服务


  1. 找到 Mysql ，这里会有两种情况，第一种服务名为 MySQL ，第二种服务名不叫 MySQL

  1. 先解决第一种情况：服务名为 MySQL
  2. 以管理员身份运行 cmd 然后输入以下指令，用来删除 MySQL 服务名
sc delete MySQL
  1. MySQL 服务名删除后，用 cd 切换到你安装 MySQL 的 bin 目录下
  2. 输入以下指令，重新指定 MySQL 的服务名
mysqld --install MySQL5    //这里指定 MySQL5 为新的名字
  1. 重启 MySQL
net start MySQL5

  1. 验证：在桌面找到控制面板 — 点击 — 在搜索栏中搜索服务 — 找到查看本地服务

  1. 此时便转变为第二种情况：MySQL 服务名不叫 MySQL
  2. 方法一：打开 phpstudy — MySQL 的配置 — 修改端口



  1. 方法二：在桌面找到控制面板 — 点击 — 在搜索栏中搜索服务 — 找到查看本地服务 — 找到 MySQL 服务 — 关闭所有服务
