Android Studio kotlin 转 Java

一. 随笔记录

java代码可以转化成kotlin代码，当然 Kotlin 反过来也可以转java

在Android Studio中 可以很方便的操作

AS 环境：Android Studio Iguana | 2023.2.1 

二. 操作步骤

1.步骤 顶部Tools ----->Kotlin ------>Show Kotlin Bytecode

步骤2：kotlin文件转为字节码 

显示如图：

步骤3：点击插件窗口左上角Decompile 按钮，就会反编译为Java文件

代码转化率90%左右，剩下的需要自己调整一下代码格式。
