🔍 一、初识操作符 %

在Python中，% 是一个常用的操作符，被称为取模运算符或求余运算符。它的作用是返回两个数相除的余数。对于整数，它直接返回余数；对于浮点数，它会先对两个数进行整数除法，然后返回余数。

  • 举个简单的例子：

    remainder = 7 % 3  # 结果为 1，因为 7 除以 3 的余数是 1
print(remainder)

在这个例子中，7 % 3 的结果是 1，因为 7 除以 3 的商是 2，余数是 1。

💡 二、操作符 % 的基本用法

操作符 % 的基本用法非常直接，就是用于求余数。这在许多场景中都非常有用，比如检查一个数是否为偶数、计算循环中的索引位置等。

  • 例如，检查一个数是否为偶数：

    def is_even(number):
    return number % 2 == 0

print(is_even(4))  # 输出 True，因为 4 是偶数
print(is_even(5))  # 输出 False，因为 5 不是偶数

在上面的代码中，我们定义了一个函数 is_even，它接受一个整数作为参数，并检查该数除以 2 的余数是否为 0。如果是
0，则这个数是偶数，函数返回 True；否则返回 False。

🎯 三、操作符 % 与字符串格式化

除了基本的数学运算，操作符 % 在Python中还有一个重要的用途，那就是字符串格式化。在较老的Python版本中，% 常常用来格式化字符串，虽然现在更推荐使用 .format() 方法或 f-string，但了解其用法仍然很有价值。

  • 代码示例

    name = "Alice"
age = 25
formatted_string = "My name is %s and I am %d years old." % (name, age)
print(formatted_string)  # 输出：My name is Alice and I am 25 years old.

在上面的例子中，%s 是一个占位符，表示一个字符串；%d 是一个占位符，表示一个整数。% 操作符后面的元组 (name, age) 提供了这些占位符的值。

🔄 四、操作符 % 的高级用法

操作符 % 还可以用于更复杂的场景，比如生成周期性的序列或者进行位运算。

  1. 生成周期性序列

    for i in range(10):
    print(i % 3)  # 输出 0, 1, 2, 0, 1, 2, ... 重复

上面的代码会打印出一个周期性序列，其中每个数都是其索引值对 3 取模的结果。这种周期性序列在很多算法和数据处理任务中都非常有用。
